# Настройка DNS для ispravleno.pro через интерфейс reg.ru

## Вариант 1: Использовать DNS серверы reg.ru (рекомендуется, проще)

Если видите в интерфейсе reg.ru:
- DNS-серверы: `ns1.hosting.reg.ru`, `ns2.hosting.reg.ru`
- Кнопка "Изменить" рядом с DNS-серверами

### Шаг 1: Управление записями через панель хостинга reg.ru

Если в разделе "Ресурсные записи" написано, что нужно перейти в панель управления хостингом:

1. **Нажмите кнопку "Изменить"** рядом с DNS-серверами
2. Или найдите раздел **"Панель управления хостингом"**
3. Или перейдите в раздел домена и найдите **"Записи зоны"** / **"DNS-записи"**

### Шаг 2: Измените A-записи в панели хостинга

В панели управления хостингом reg.ru найдите раздел **"DNS-записи"** или **"Записи зоны"** и настройте:

1. **A-запись для @ (корневой домен):**
   - Имя: `@` (или пустое)
   - Тип: `A`
   - Значение: `212.74.227.208`
   - TTL: `3600`

2. **A-запись для www:**
   - Имя: `www`
   - Тип: `A`
   - Значение: `212.74.227.208`
   - TTL: `3600`

3. **Сохраните изменения**

### Шаг 3: Альтернатива - через кнопку "Изменить"

Попробуйте нажать кнопку **"Изменить"** рядом с DNS-серверами - возможно, там есть возможность управления записями.

---

## Вариант 2: Изменить DNS-серверы на внешние (если есть свой DNS)

Если у вас есть свой DNS сервер или вы хотите использовать внешний сервис (например, Cloudflare):

1. **Нажмите кнопку "Изменить"** рядом с DNS-серверами
2. **Измените DNS-серверы** на ваши (например, Cloudflare: `ns1.cloudflare.com`, `ns2.cloudflare.com`)
3. **Настройте A-записи** на выбранном DNS-сервисе:
   - `@` → `212.74.227.208`
   - `www` → `212.74.227.208`

---

## Вариант 3: Если нет доступа к панели хостинга

Если домен не привязан к хостингу reg.ru и нет доступа к управлению записями:

### Решение через изменение DNS-серверов на внешние

1. Используйте бесплатный DNS-сервис (например, Cloudflare):
   - Зарегистрируйтесь на https://www.cloudflare.com
   - Добавьте домен `ispravleno.pro`
   - Получите DNS-серверы (например, `ns1.cloudflare.com`, `ns2.cloudflare.com`)

2. В reg.ru нажмите **"Изменить"** рядом с DNS-серверами

3. Введите DNS-серверы от Cloudflare:
   - `ns1.cloudflare.com`
   - `ns2.cloudflare.com`

4. Сохраните изменения

5. В Cloudflare настройте A-записи:
   - `@` → `212.74.227.208`
   - `www` → `212.74.227.208`

---

## Рекомендуемый вариант (самый простой)

Если у вас нет доступа к панели хостинга reg.ru, **используйте Cloudflare** (бесплатно):

1. Регистрация на Cloudflare: https://dash.cloudflare.com/sign-up
2. Добавьте домен `ispravleno.pro`
3. Получите DNS-серверы от Cloudflare
4. В reg.ru измените DNS-серверы на Cloudflare
5. В Cloudflare настройте A-записи на `212.74.227.208`

**Преимущества Cloudflare:**
- ✅ Бесплатный
- ✅ Быстрая настройка
- ✅ Дополнительная защита
- ✅ Бесплатный SSL

## После настройки DNS

Когда DNS записи будут указывать на `212.74.227.208`, выполните на вашем сервере:

```bash
cd ~/Ispravleno/ispravleno-website/website
git pull origin main
chmod +x setup-ispravleno-pro.sh
./setup-ispravleno-pro.sh
npm run build
pm2 restart ispravleno-website
```

## Проверка

После настройки проверьте:
```bash
dig ispravleno.pro
# или
nslookup ispravleno.pro
```

Должно вернуться: `212.74.227.208`
